Типи даних:

  • Числа (Numbers):

    Включає цілі числа та числа з плаваючою комою, наприклад: 5, 3.14, -10

  • Рядки (Strings):

    Використовуються для представлення текстової інформації. Рядки повинні бути заключені в подвійні (") або одинарні (') лапки, наприклад: "Hello, world!", 'JavaScript'.

  • Булеві значення (Booleans):

  • Мають два можливих значення: true (істина) і false (неправда). Вони використовуються для логічних виразів.

  • Масиви (Arrays):

    Використовуються для зберігання колекцій даних. Масив може містити різні типи даних та інші масиви.

  • Об'єкти (Objects):

    Використовуються для зберігання колекцій ключ-значення, де ключі - це рядки, а значення - будь-які типи даних.

  • Null:

    Представляє відсутність значення або непевність щодо значення.

  • Undefined:

    Вказує на те, що змінна була оголошена, але їй не було призначено значення.

  • Символи (Symbols):

    Це унікальні імутабельні значення, які можна використовувати як ключі для властивостей об'єктів

  • Функції (Functions):

    В JavaScript функції є об'єктами, і вони можуть бути використані для виконання коду та повернення значень.

Вирази в JavaScript

Kомбінації операндів і операторів:

  • Арифметичні вирази:

    const sum = 5 + 3; // Додавання
    const difference = 10 - 4; // Віднімання
    const product = 6 * 2; // Множення
    const quotient = 8 / 2; // Ділення

  • Логічні вирази:

    const isTrue = true;
    const isFalse = false;
    const logicalAnd = isTrue && isFalse; // Логічне "І"
    const logicalOr = isTrue || isFalse; // Логічне "АБО"

  • Рядкові вирази (конкатенація рядків):

    const firstName = "John";
    const lastName = "Doe";
    const fullName = firstName + " " + lastName; // Конкатенація рядків

  • Вирази зі змінними та функціями:

    const x = 5;
    const y = 10;
    function add(a, b) { return a + b; }
    const result = add(x, y); // Виклик функції та обчислення

  • Вирази з логічними умовами:

    const age = 25;
    const hasLicense = true;
    const canDrive = age >= 18 && hasLicense; // Логічна умова

  • Вирази з масивами та об'єктами:

    const numbers = [1, 2, 3, 4, 5];
    const person = { name: "Alice", age: 30 };
    const firstNumber = numbers[0]; // Отримання першого елементу масиву
    const personName = person.name; // Отримання властивості об'єкта
