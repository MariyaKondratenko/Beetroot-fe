Mожливі варіанти створення функцій:

  • Function Declaration (Оголошення функції):

    function greet(name) {
    console.log(`Привіт, ${name}!`); }

  • Function Expression (Вираз функції):

    const greet = function(name)
    { console.log(`Привіт, ${name}!`); };

  • Arrow Function (Стрілкова функція):

    const greet = (name) => {
    console.log(`Привіт, ${name}!`); };

  • Anonymous Function (Анонімна функція):

    const greet = function(name)
    { console.log(`Привіт, ${name}!`); };

  • Immediately Invoked Function Expression (IIFE):

    (function(name) {
    console.log(`Привіт, ${name}!`); })("Олег");

  • Named Function Expression (Іменована вираз функції):

    const greet = function sayHello(name) {
    console.log(`Привіт, ${name}!`); };
